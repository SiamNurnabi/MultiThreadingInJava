What is a thread?
A thread is a lightweight sub-process, the smallest unit of processing. It is a separate path of execution.
Threads are independent, if an exception occurs in one thread, it doesn't affect other threads. It shares a common memory area.

What is a job in thread?
A thread is a job that is executed by the CPU. When a thread is created, a job is assigned to it.
The job is nothing but the code that is to be executed by the thread.

What is a process?
A process is a program in execution. A process is an independent entity to which system resources (CPU time, memory, etc.) are allocated.
